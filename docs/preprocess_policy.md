# Preprocessing Policy

## 1. Що очищаємо
- Зайві пробіли та табуляції (`\s+`)
- Переноси рядків, пусті рядки
- HTML-артефакти (теги `<.*?>` та HTML-сутності, наприклад `&nbsp;`)
- Фото- та соц-кредити (наприклад, "Фото Укрінформ", "Допис, поширений @…")

## 2. Що нормалізуємо
- Лапки: (`«…»`, `“…”`) → `"…"`
- Тире та дефіси: (`–`, `—`) → `-`
- Нерозривні пробіли (non-breaking space)
- Заміна гомогліфів (наприклад, латинська "a" → українська "а")

## 3. Що маскуємо
- URL → `<URL>`
- Email → `<EMAIL>`
- Телефон → `<PHONE>`

## 4. Що не чіпаємо
- Числа, версії, дати, час (наприклад, `3.14`, `v1.2.3`, `18:00`)
- Спеціальні коди або скорочення, які важливі для семантики тексту

## 5. Sentence split
- Використовуємо регулярні вирази з урахуванням скорочень:
  - `м.`, `вул.`, `р.`, `т.д.`, `т.ін.`, `рр.`, `т.п.`, `ім.` 
  - Числа, версії, порядкові номери у списках (`3.14`, `v1.2.3`, `1.`) не розбиваються
- Ділимо речення за символами `.`, `!`, `?`, якщо вони **не є** частиною абревіатур або чисел

## 6. Приклади "до / після"
