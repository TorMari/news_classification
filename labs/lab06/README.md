# Lab 6 — TF-IDF + Logistic baseline

## 1. Задача

**Тип задачі:** А — класифікація текстів (тема/тональність/спам/urgency) 

---

## 2. Classification-підзадача

Встановлення теми новин (культура, спорт, економіка, політика) на основі змісту їхніх текстів.

---

## 3. Baseline comparison: processed text vs lemma text

Було реалізовано базову модель класифікації на основі Logistic Regression у поєднанні з TfidfVectorizer. Для даного корпусу новин не спостерігалося суттєвої різниці в якості класифікації між використанням обробленого тексту (processed text) та лематизованого тексту (lemma text), оскільки обидва підходи продемонстрували подібні значення метрик accuracy та macro-F1.

---

## 4. Основні метрики

Accuracy (processed text): 0.96
Macro-F1 (processed text): 0.9598660472319009
Accuracy (lemma text): 0.96
Macro-F1 (lemma text): 0.9602176375278031

---

## 5. Error analysis та подальші покращення

Найчастіше зустрічаються помилки типу **overlap класів** та **домінування сильних маркерів**, що свідчить про складність текстів із змішаною тематикою та залежність моделі від окремих вагомих слів. Також помітною є проблема недостатнього врахування контексту, особливо для складених понять.

У першу чергу доцільно покращити обробку ознак: додати або посилити використання n-грам (біграм), щоб модель краще розпізнавала сталі словосполучення. Другим кроком варто зменшити вплив надто частих або “сильних” маркерів (наприклад, числових показників чи політичних назв), щоб уникнути зміщення класифікації.
